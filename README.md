# Bias-Aware Skin Disease Synthesis Pipeline

## Overview

### Problem Statement

The datasets for skin diseases highly under-represent darker skin tones, which in turn increases the risks of false negatives and late detection. We aim to develop a pipeline that is reproducible and can be used across different skin diseases. Although the initial project would be focused on melanoma datasets.

### Objectives

To solve the problem, we plan to develop a pipeline that is reproducible and can be used across different skin diseases. Although the initial project would be focused on melanoma datasets.

## Setup

### 1. Clone the Repository

```bash
git clone https://github.com/Muneeb-Ahmad-404/bias-aware-stylegan  
cd bias-aware-stylegan 

```

### 2. Create Virtual Environment (using uv)

```bash
uv venv
source .venv/bin/activate   # Linux/Mac
.venv\Scripts\activate      # Windows

```

### 3. Install Requirements (using uv)

```bash
uv pip install -r requirements.txt

```

### 4. Fetch the Dataset via DVC (after remote is configured)

```bash
dvc pull

```
